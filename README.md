# AFARQ Arquitectura + Construcción

Bienvenido a AFARQ Arquitectura + Construcción, un proyecto dedicado a proporcionar soluciones innovadoras y sostenibles en el campo de la arquitectura y la construcción. Nuestro objetivo es transformar espacios, optimizar recursos y generar un impacto positivo en las comunidades a través de un diseño consciente y funcional.

## Descripción

AFARQ es una plataforma donde se pueden encontrar nuestras propuestas arquitectónicas, proyectos en construcción y servicios relacionados. Nuestro enfoque se centra en la colaboración con nuestros clientes para crear espacios que se adapten a sus necesidades.

## Características

- Diseño arquitectónico personalizado.
- Gestión de proyectos de construcción.
- Asesoramiento en selección de materiales sostenibles.
- Publicación de artículos y estudios de caso sobre tendencias en arquitectura y construcción.

## Tecnologías Utilizadas

- [WordPress](https://wordpress.org/)
- PHP
- HTML / CSS
- JavaScript
- MySQL

## Requisitos

Para ejecutar este proyecto, necesitas tener instalado:

- PHP 7.4 o superior
- MySQL 5.7 o superior
- Un servidor web como Apache o Nginx
- Composer (para la gestión de dependencias, si se usa)
